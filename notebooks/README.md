# Notebooks

Thư mục này lưu các notebook phân tích và template Papermill.

Gợi ý notebook nên tạo trong giai đoạn build:
- `01_data_quality_check.ipynb`: profile dữ liệu ingest, tỷ lệ null, phân phối giá trị.
- `02_price_trend_analysis.ipynb`: phân tích xu hướng giá theo thời gian/khu vực.
- `03_pipeline_kpi_report.ipynb`: báo cáo KPI vận hành pipeline theo từng run.
