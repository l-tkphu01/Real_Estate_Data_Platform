# Notebooks

Thư mục này lưu các notebook phân tích dữ liệu Bất Động Sản.

Gợi ý notebook nên tạo:
- `01_data_quality_check.ipynb`: profile dữ liệu ingest, tỷ lệ null, phân phối giá trị.
- `02_price_trend_analysis.ipynb`: phân tích xu hướng giá BĐS theo thời gian/khu vực.
- `03_pipeline_kpi_report.ipynb`: báo cáo KPI vận hành pipeline theo từng run.
